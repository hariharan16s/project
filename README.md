## Base docker image used

ubuntu:latest

## Installation

1. Install Docker
2. Build the docker image from the github Dockerfile
   `docker build -t hari_project github.com/hariharan16s/project`

## Usage

#### Run hari_project docker container

`docker run -ti --rm hari_project`
